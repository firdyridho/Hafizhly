# 📖 Hifzly - Aplikasi Pendamping Murojaah Al-Qur'an

**Hifzly** adalah aplikasi web berbasis *Artificial Intelligence* yang dirancang untuk membantu umat Muslim dalam menjaga, memperbaiki, dan memantau hafalan Al-Qur'an mereka. Dengan fitur AI-powered voice recognition dan algoritma Spaced Repetition System (SRS), Hifzly menjadi teman setia dalam perjalanan menghafal Al-Qur'an secara mandiri.

---

## ✨ Fitur Utama

### 1. Smart Murojaah (AI Powered)
*   **Voice Recognition:** Setoran hafalan secara *real-time* menggunakan AI yang mendengarkan lafalan ayat pengguna.
*   **Auto-Reveal Text:** Teks ayat akan terbuka otomatis saat pengguna membacanya dengan benar.
*   **Tap to Skip:** Fitur interaktif untuk melewati kata jika AI mengalami kendala deteksi.

### 2. Spaced Repetition System (SRS)
*   Algoritma pintar untuk menjadwalkan murojaah. Pengguna bisa memberikan rating (Sulit/Lancar/Mudah) yang menentukan kapan ayat tersebut harus diulangi kembali.

### 3. Mushaf Interaktif
*   Tampilan ayat yang nyaman dibaca dengan font *Scheherazade New* untuk keakuratan simbol waqaf.
*   Terjemahan per ayat yang bisa disembunyikan/ditampilkan.
*   Tafsir Kemenag RI per ayat (on-demand).
*   Audio Murottal (Full Surah & Per Ayat).

### 4. Mutabaah (Tracking)
*   Kalender aktivitas untuk memantau konsistensi tilawah dan hafalan.
*   Statistik harian, mingguan, dan *streak* motivasi.

### 5. Doa Harian
*   Koleksi doa harian lengkap dengan teks Arab, Latin, Terjemahan, dan visual yang menarik.

---

## 🛠️ Teknologi yang Digunakan

*   **Frontend:** HTML5, CSS3, JavaScript (ES6+), FontAwesome
*   **Backend:** PHP (Native)
*   **Database:** MySQL
*   **APIs:** 
    *   [EQuran.id API](https://equran.id/apidev) (Surah, Ayat, Tafsir)
    *   [Quran Gading Dev API](https://api.quran.gading.dev/) (Juz)
    *   Web Speech API (untuk AI Voice Recognition)
